# ReMol
![image](img/ReMol.png)


## Install environment

**1. GPU environmentx**<br>  
CUDA 11.1

**2. create a new conda environment**<br>  
conda create -n ReMol python=3.7<br>  
conda activate ReMol  

**3. download some packages**<br>  
pip install -r requirements.txt<br>  
source activate ReMol
